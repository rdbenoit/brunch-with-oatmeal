brunch-with-oatmeal
===================

### Included Framworks
* Twitter Bootstrap
* Font Awesome
* jQuery
* Underscore
* Backbone
* Backbone Layout Manager

### Getting Started
1. Install Brunch `npm install -g brunch`
2. Create Brunch with Oatmeal project `brunch new <project name> -s git@github.com:rdbenoit/brunch-with-oatmeal.git`
